# How to Use the Presentation Files

## Files Created:
1. **PROJECT_PRESENTATION.md** - Markdown version (for reference)
2. **PROJECT_PRESENTATION.html** - HTML presentation (view in browser)
3. **CREATE_PPT_INSTRUCTIONS.md** - Instructions for creating PPT

---

## Option 1: View HTML Presentation (Easiest)

1. **Open the HTML file:**
   - Double-click `PROJECT_PRESENTATION.html`
   - It will open in your default browser

2. **Navigate:**
   - Scroll down to see all slides
   - Use browser zoom (Ctrl + / Ctrl -) to adjust size

3. **Export to PDF:**
   - Press `Ctrl + P` (or Cmd + P on Mac)
   - Select "Save as PDF"
   - Choose "More settings" → "Background graphics" (to keep colors)
   - Click "Save"

4. **Convert PDF to PPT:**
   - Use online tools like:
     - https://www.ilovepdf.com/pdf-to-ppt
     - https://www.adobe.com/acrobat/online/pdf-to-ppt.html
   - Upload the PDF
   - Download the PPT file

---

## Option 2: Create PowerPoint Manually (Best Quality)

1. **Open Microsoft PowerPoint**

2. **For each slide in PROJECT_PRESENTATION.md:**
   - Create a new slide
   - Copy the title as the slide title
   - Copy the content as bullet points
   - Add relevant images/icons

3. **Design Tips:**
   - Use dark theme (matches the HTML)
   - Add orange/red accents for highlights
   - Use monospace font for code/technical content
   - Add screenshots from your project

4. **Add Images:**
   - Dashboard screenshot
   - Transaction analysis result
   - Link checker interface
   - Deepfake detection result
   - System architecture diagram

---

## Option 3: Use Online Converters

### Markdown to PPT:
1. Go to https://www.markdowntopresentation.com/
2. Upload `PROJECT_PRESENTATION.md`
3. Download the PPT

### HTML to PPT:
1. Go to https://www.zamzar.com/convert/html-to-pptx/
2. Upload `PROJECT_PRESENTATION.html`
3. Download the PPT

---

## Option 4: Use Presentation Software

### Google Slides:
1. Open Google Slides
2. File → Import → Upload
3. Upload the HTML file or copy content from markdown
4. Download as PowerPoint

### LibreOffice Impress:
1. Open LibreOffice Impress
2. File → Open → Select HTML file
3. File → Export as PDF → Convert to PPT

---

## Recommended Approach:

**For Quick Demo:**
- Use the HTML file directly in browser
- Full-screen mode (F11)
- Navigate with arrow keys or scroll

**For Professional Presentation:**
1. Open HTML in browser
2. Export to PDF (Ctrl + P)
3. Convert PDF to PPT using online tool
4. Edit in PowerPoint to add:
   - Your screenshots
   - Company logo
   - Custom animations
   - Your branding
